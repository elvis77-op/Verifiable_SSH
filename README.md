## steps to deploy verifiable_ssh:
    1. run "git clone https://github.com/elvis77-op/Verifiable_SSH.git"
    2. run "chmod +x *.sh"
    3. define the address and port of TDVM in main.sh
    4. run "./build_vssh.sh"
    5. run "./launch_vssh.sh"

## to remove the keys of veriafiable_ssh, run "./clean_vssh.sh"
