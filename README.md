## steps to deploy verifiable_ssh:
    1. git clone https://github.com/elvis77-op/Verifiable_SSH.git 
    2. define the address and port of TDVM in main.sh
    3. run "./build_vssh.sh"
    4. run "./launch_vssh.sh"

## to remove the keys of veriafiable_ssh, run "./clean_vssh.sh"
