## steps to deploy verifiable_ssh:
    1. run "git clone https://github.com/elvis77-op/Verifiable_SSH.git"
    2. run "chmod +x *.sh" 
    3. add proxy in Dockerfile (optional)
    4. define the address and port of TDVM in main.sh
    5. run "./build_vssh.sh"
    6. run "./launch_vssh.sh"

## environment clean up
    run "./clean_vssh.sh"
